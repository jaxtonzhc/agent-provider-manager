## 2. Sync Flow

```
                           ┌─────────────────┐
                           │ apm sync <name> │
                           │ [--agents ...]  │
                           │ [--dry-run]     │
                           └────────┬────────┘
                                    │
                                    ▼
                           ┌─────────────────┐

                   ┌────────────┐      ┌────────────┐

              ┌─────────────────────┐

                         │                                  │

                         │                                  │
```

## 3. Provider Resolution Flow

```
                    ┌─────────────────────────┐
                    │ apm provider add <name> │
                    │ --key <key>             │
                    │ [--variant <variant>]   │
                    └────────────┬────────────┘
                                 │
                                 ▼
                    ┌─────────────────────────┐
                    │ resolve_provider()      │
                    │ from registry.py        │
                    └────────────┬────────────┘
                                 │
                                 ▼
                    ┌─────────────────────────┐
                    │ Load registry           │
                    │ (cache > remote >       │
                    │  built-in)              │
                    └────────────┬────────────┘
                                 │
                                 ▼
                    ┌─────────────────────────┐
                    │ Find provider by name   │
                    │ in registry.providers   │
                    └────────────┬────────────┘
                                 │
                       ┌─────────┴─────────┐
                       │                   │
                       ▼                   ▼
                ┌────────────┐      ┌────────────────┐
                │  Found     │      │ Not Found      │
                │            │      │ → Custom mode  │
                └─────┬──────┘      │   Need --url   │
                      │             └────────────────┘
                      ▼
           ┌─────────────────────┐
           │ Check variants      │
           │ (token-plan-cn,     │
           │  api, etc.)         │
           └──────────┬──────────┘
                      │
            ┌─────────┴─────────┐
            │                   │
            ▼                   ▼
     ┌────────────┐     ┌────────────────┐
     │ Variant    │     │ No variant     │
     │ specified  │     │ → Use first    │
     │ → Use it   │     │   + show list  │
     └─────┬──────┘     └────────┬───────┘
           │                     │
           └──────────┬──────────┘
                      │
                      ▼
           ┌─────────────────────┐
           │ Build provider dict │
           │ {                   │
           │   name,             │
           │   base_url,         │
           │   api_key,          │
           │   protocol,         │
           │   models            │
           │ }                   │
           └──────────┬──────────┘
                      │
                      ▼
           ┌─────────────────────┐
           │ Save to             │
           │ ~/.apm/providers    │
           │ .json               │
           └─────────────────────┘
```

## 4. Registry Hybrid Resolution

```
                    ┌─────────────────┐
                    │ load_registry() │
                    └────────┬────────┘
                             │
                             ▼
                    ┌─────────────────┐
                    │ Check cache     │
                    │ ~/.apm/registry │
                    │ -cache.json     │
                    └────────┬────────┘
                             │
                   ┌─────────┴─────────┐
                   │                   │
                   ▼                   ▼
            ┌────────────┐      ┌────────────┐
            │ Cache hit  │      │ Cache miss │
            │ (< 24h)    │      │ or expired │
            │ → Return   │      │            │
            └────────────┘      └─────┬──────┘
                                      │
                                      ▼
                             ┌─────────────────┐
                             │ Fetch remote    │
                             │ GitHub raw JSON │
                             └────────┬────────┘
                                      │
                            ┌─────────┴─────────┐
                            │                   │
                            ▼                   ▼
                     ┌────────────┐      ┌────────────┐
                     │ Success    │      │ Failed     │
                     │ → Cache it │      │ (offline)  │
                     │ → Return   │      │            │
                     └────────────┘      └─────┬──────┘
                                               │
                                               ▼
                                      ┌─────────────────┐
                                      │ Use built-in    │
                                      │ registry.json   │
                                      │ (shipped with   │
                                      │  package)       │
                                      └─────────────────┘
```
